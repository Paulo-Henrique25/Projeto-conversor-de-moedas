<p>Este projeto apresenta um site totalmente responsivo, desenvolvido para oferecer uma experiência consistente em diferentes dispositivos.  
O layout se adapta de forma fluida a telas de desktop, tablets e smartphones, garantindo navegação intuitiva e design moderno.  
Inclui seções de destaque, botões interativos e uma interface otimizada para usabilidade e performance.</p>
